Portfolio Website

Website Address: http://jonathans80.sg-host.com/

Website Platform: WordPress

Website Purpose: The purpose of my website is to provide potential clients with my extensive portfolio. Included in this webpage will be my experience with programming languages such as HTML, CSS, C#, C++, Javascript, and Python. All smaller coding projects will have their source code attached all programmed and designed by me.

Audience: The audience I am targeting for my website will be potential employers and tech enthusiasts. The site will showcase my technological portfolio for those interested in subjects of programming, and games.

3 New Techniques:

  • I've learned how to showcase my source code in an appealing format using the plugin code blocks. I used the code blocks plugin to display my source code for single script projects like Obj Reader, Pong, and Snake. A very useful tool for a programmer to showcase their programming fluency.
  • I've learned advanced techniques using the section elements from the otter plugin to create the side-by-side image text formats. I used it to display my image and some text side by side more efficiently. An example of my usage would be the about page where I feature an image of myself and some resume text to the side.
  • I've learned how to use custom HTML blocks inside of WordPress to create fancy links to my games. I used this feature to ad custom-made widgets from Itch.io to my website. The games I used this for are Colorless, Passing Flame (pre and post-jam), Rock Slide, UFO, and Blizzard.
